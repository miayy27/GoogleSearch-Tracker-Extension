# GoogleSearch-Tracker-Extension

This repository contains the source code for the custom Google Chrome extension developed for the empirical study: **"Cognitive Style Effects in Traditional and Conversational Search"**.

## Description
This extension is designed to capture and analyze user web search behaviors in traditional Information Retrieval (IR) systems (e.g., Google). It is a lightweight, non-intrusive tool that integrates directly into the browsing environment to enable real-time and retrospective tracking of search interactions without requiring external screen recording or third-party monitoring software.

### Key Features:
* **Automated Data Logging**: Meticulously records critical search metrics, including User ID, date, duration of interaction, URLs visited, and specific search terms used.
* **Data Export**: Allows researchers to easily download the captured search session logs in `JSON` format for offline qualitative content analysis and quantitative statistical modelling.
* **Privacy-Focused**: Designed for controlled laboratory experiments; it only tracks data relevant to the assigned search tasks.
